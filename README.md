# Expense Tracker Backend

A backend REST API built using Spring Boot for managing expenses.

## Features
- Add expense
- View all expenses
- Update expense
- Delete expense

## Tech Stack
- Java
- Spring Boot
- REST APIs

## API Endpoints

| Method | Endpoint | Description |
|--------|---------|-------------|
| POST | /expenses | Add expense |
| GET | /expenses | Get all expenses |
| PUT | /expenses/{id} | Update expense |
| DELETE | /expenses/{id} | Delete expense |

## Status
Currently uses in-memory storage. Database integration coming soon.
